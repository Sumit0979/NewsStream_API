# NewsStream – API Based News Platform

NewsStream is a responsive web application that displays real-time news using an external news API. The platform fetches the latest news articles and presents them in a clean and user-friendly interface.

##  Features

- Fetches real-time news using the **NewsData API**
- Responsive design for desktop and mobile devices
- Category-based news filtering
- Dynamic content rendering using JavaScript
- Clean and simple user interface
- Fast loading and easy navigation

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **NewsData API**

## How It Works

1. The application sends a request to the **NewsData API**.
2. The API returns the latest news articles in JSON format.
3. JavaScript processes the data and dynamically displays the news on the webpage.
4. Users can filter news based on different categories.
